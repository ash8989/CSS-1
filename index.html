What is CSS?
  1. CSS stands for Cascading Style Sheet.

  2. CSS describes how HTML elements are to be displayed on the screen,paper or in other media.

  3. Css is not necessary CSS saves a lots of work.It can control the layout of multiple webpages all at once.

  4. External stylesheets are stored in CSS files.

CSS can be added to HTML into 3 ways:
  • Inline - By using the style attribute inside HTML elements1.
  • Internal - By using the style element in the head section.
  • External - By using link element to link to an external CSS files.
  • The most common way to add CSS , is to keep the styles in th external CSS files. However , in this tutorial we will use inline and internal CSS styles , because it is easier to demonstrate , and easier for you to try it yourself.

When comparing CSS CLASS vs ID , the difference is that CSS CLASS can applies a styles to multiple elements. On the other hand , CSS ID can applies a style to one unique element . ID is also special in that you can use a special URL to link directly to an elemenet and it's used by Javascript.

In CSS , selector are used to target a specific element or range of elements on the webpage . Once an element has been targeted , a style or a set of styles can be applied to the elements.

There is a wide range of selectors available . Two of the most commonly used selectors are CLASS and ID . Both are used to target elements to which a style to be applied.
